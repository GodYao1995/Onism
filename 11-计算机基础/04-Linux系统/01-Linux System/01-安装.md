#### yum

RPM(红帽软件管理器)是为了简化安装的复杂度，而Yum软件仓库是为了解决软件包之间的依赖关系

~~~bash
yum install epel-release # Extra Packages for Enterprise Linux
yum repolist all
yum list all
yum info package
yum install package
yum reinstall package
yum update package
yum remove package
yum clean all
~~~
