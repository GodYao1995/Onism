#### Hello World

> 新建文件 HelloWorld.java 文件名和类名一致

~~~java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
~~~

运行

~~~bash
javac HelloWorld.java
java HelloWorld
~~~
